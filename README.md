# 🚀 DigiTools – Premium Digital Marketplace

DigiTools is a modern and responsive web application where users can explore premium digital products and manage their purchases through an intuitive cart system.

---

## 📖 Description
DigiTools is a curated marketplace for high-quality digital assets and subscription-based tools. Designed with a focus on UI/UX precision, this project transforms a complex Figma design into a pixel-perfect, functional web experience. It ensures a seamless user journey, whether you are browsing tools on a mobile phone or a desktop.

---

## 🛠️ Technology Stack
The project is built using modern web development tools to ensure performance and scalability:
1. React.js: For building a component-based, dynamic user interface.
2. Tailwind CSS: For rapid, customized, and responsive styling.
3. daisyUI: For enhanced UI components and clean, accessible design elements.
4. JavaScript (ES6+): For handling application logic and state management.
5. React Toastify: For providing real-time, interactive user feedback and notifications.

---

## ✨ Key Features

### 1. 🛒 Smart "Add to Cart" System
Users can easily add tools to their shopping cart with a single click. The system includes built-in validation to prevent duplicate entries—if a user tries to add the same item twice, it triggers a warning notification to ensure a smooth experience.

### 2. ❌ Effortless Cart Management
The application features a dedicated cart view where users can review their selected items. If they change their mind, they can instantly remove specific products from the list, making the management of digital tools quick and hassle-free.

### 3. 💰 Dynamic Price Calculation
As users add or remove items, the application automatically calculates and updates the total price in real-time. This provides users with an accurate summary of their potential costs before proceeding to checkout.
